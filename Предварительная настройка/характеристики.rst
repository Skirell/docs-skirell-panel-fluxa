Технические характеристики
===========================

Питание
-------

.. list-table::
   :widths: 25 75
   :header-rows: 0

   * - **Напряжение питания**
     - 48 В через PoE 802.3af; переменное напряжение 110 ~ 240 V, 50 Hz
   * - **Потребляемая мощность**
     - 12 Вт
   * - **АКБ**
     - Нет

Железо
------

.. list-table::
   :widths: 25 75
   :header-rows: 0

   * - **Экран**
     - 4 дюймовый IPS экран, разрешение 480x480 пикселей
   * - **Тачскрин**
     - Есть
   * - **Процессор**
     - Rockchip RK3566 Quad-core Cortex-A55, 1.8 ГГц
   * - **RAM**
     - 2 Гбайта
   * - **eMMC**
     - 32 Гбайт

Интерфейсы
----------

.. list-table::
   :widths: 25 75
   :header-rows: 0

   * - **Ethernet**
     - RJ45-100M, с поддержкой PoE 802.3af
   * - **RS-485**
     - Есть
   * - **Wi-Fi**
     - 802.11b/g/n
   * - **Bluetooth**
     - Версия 5.2
   * - **USB**
     - Type-C
   * - **Реле**
     - Отсутствует
   * - **Дополнительно**
     - Два микрофона, динамик, датчик освещённости

Программное обеспечение
-----------------------

.. list-table::
   :widths: 25 75
   :header-rows: 0

   * - **Операционная система**
     - Android 11
   * - **Приложения**
     - Веб-браузер и десятки приложений из Google Play
   * - **Интеграция с контроллером Wiren Board**
     - Поддержка Ethernet и Wi-Fi; веб-браузер для вывода веб-интерфейса контроллера или SCADA-системы; приложения от производителей ПО для автоматизации и SCADA-систем

Условия эксплуатации
--------------------

.. list-table::
   :widths: 25 75
   :header-rows: 0

   * - **Температура воздуха**

   * - **Гарантийный срок**

   * - **Срок службы**
     - 5 лет
