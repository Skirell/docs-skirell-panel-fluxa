Подключение панели
=========================

Для начала управления системой умного дома при помощи панели, необходимо выполнить следующие действия:

1.	Подключить панель к питанию.
2.	Подключить панель к сети (WiFi – беспроводная связь; Ethernet – проводная связь);
3.	Подключить панель к MQTT-брокеру;
4. Загрузить готовый JSON-файл в панель.

Подключение к питанию
----------------------

Панель имеет возможность подключения к питанию двумя способами:

* :ref:`Через PoE` (Power over Ethernet), т. е. используя провод Ethernet.
* :ref:`Через AC` (Alternating Current), т. е. с используя провод переменного тока.

Подключение к сети
--------------------

При запуске панели отображается первая страница домашнего экрана. Для перехода к общим настройкам нужно потянуть верхнюю часть экрана вниз. Старайтесь ориентироваться на верхнюю область, показанную на рисунке ниже.

.. image:: /images/setting/main.png

Страница общих настроек содержит в себе информативный блок, который отображает:

   * версию прошивки панели
   * IP-адрес панели
   * MAC-адрес панели. MAC-адрес прогружается только 
после подключения панели к сети.

.. image:: /images/setting/main_settings.png

Ниже информационного блока располагаются кнопки:

   * Верхняя левая – для перезагрузки панели. Нажав на нее, панель моментально начнет перезагрузку.
   * Верхняя правая – для отображения состояния сети. Нажав на нее, панель вычислит скорость сети. При ошибке вычисления на кнопке высветиться соответствующее сообщение.
   * Нижняя левая – для перехода к настройкам MQTT. Нажав на нее, вы перейдете на страницу настроек параметров для подключения к MQTT-брокеру.
   * Нижняя правая – для перехода к настройкам сети. Нажав на нее, вы перейдете на страницу настроек параметров для подключения к WiFi.

Также в верхнем левом углу располагается кнопка «Назад». Нажав на нее, вы вернетесь на предыдущую страницу.

Для подключения к сети перейдите к настройкам сети.

.. image:: /images/setting/network_settings.png

На странице настроек сети находятся поля для ввода имени сети WiFi и пароля, переключатель подключения к WiFi, кнопка сохранения примененных изменений в настройках.

При подключении к сети, важно учесть следующие моменты:

    .. warning::
        В настоящий момент подключение к сети поддерживается только через Ethernet провод. Панель автоматически считывает DCHP через провод. В данном случае дополнительные данные для подключения вводить нет необходимости. Важно лишь перевести переключатель в состояние «Выкл».

Для статического подключения к WiFi в полях ввода введите данные о сети и пароль. Сохраните настройки кнопкой, располагаемой внизу страницы. Переведите переключатель в состояние «Вкл». 

Подключение к MQTT брокеру
------------------------------

Для подключения к MQTT-брокеру перейдите к настройкам MQTT.

.. image:: /images/setting/mqtt_settings.png

На странице настроек MQTT находятся поля для ввода IP – адреса, порта, логина и пароля MQTT-брокера.

Для подключения к MQTT-брокеру в полях ввода введите данные о брокере, логин и пароль при необходимости. Сохраните настройки кнопкой, располагаемой внизу страницы. После сохранения параметров интерфейс панели перезагрузится.
